# Install 

```
chmod +x install
./install
```

or

copy [Darkula.colorscheme](https://github.com/cetver/kde-konsole-darcula-theme/raw/master/Darkula.colorscheme) to `$HOME/.local/share/konsole/Darkula.colorscheme`
